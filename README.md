# snake

Каркас игры Snake на C++20 с использованием SDL2 и Lua.

## Сборка и запуск

Инструкции по установке зависимостей и сборке находятся в [docs/build_run.md](docs/build_run.md).

## Текущее состояние

- При запуске открывается окно 800x600 с заголовком "snake".
- Фон очищается однотонным цветом, окно закрывается по крестику или по клавише Esc.
- Lua подключён и печатает `Lua OK` при старте.
