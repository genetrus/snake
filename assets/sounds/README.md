# Sound placeholders

Generated placeholder beeps, CC0, 44.1kHz mono 16-bit PCM.
Binary WAV files are added locally (outside Codex) and pushed to the same codex/ branch via PowerShell.

## File mapping
- `eat.wav`: play when the snake eats food.
- `bonus_score.wav`: play when bonus score is awarded.
- `bonus_slow.wav`: play when the snake is slowed by a bonus.
- `pause_on.wav`: play when pausing the game.
- `pause_off.wav`: play when unpausing the game.
- `menu_click.wav`: play on menu selection click.
- `game_over.wav`: play when the game ends.
- `restart.wav`: play when restarting a run.
